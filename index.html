마이크로바이옴 원료 개발 Ideation 리포트
— Next-Microbiome 전략

화장품 마이크로바이옴 시장은 2025년 $430M에서 2035년 $1.2B 규모로 성장 중이다. 1세대(사균체·발효여과물) → 2세대(포스트바이오틱 복합계) → 3세대(세균 세포외소포체·정밀발효·합성생물학·피부 생태계 플랫폼)로의 전환이 가속되고 있다. 본 리포트는 기존 마이크로바이옴 원료 체계를 세대별로 정리하고, IFSCC 2025 이후 열린 'Next-Microbiome' 원료 개발 방향 6대 컨셉을 이데이션한다.

1. 화장품에서 마이크로바이옴의 개념 재정의

1.1 전통적 정의 → 확장된 정의

[기존 정의 (2015–2022)]
마이크로바이옴 화장품 = 프로바이오틱스(유산균)를 넣거나
                        마이크로바이옴을 '손상시키지 않는' 마이크로바이옴-프렌들리 제품

[확장된 정의 (2023–현재)]
마이크로바이옴 화장품 = 피부 생태계(skin ecosystem)를 능동적으로 '설계·조율'하는
                        원료 및 제형 기술 플랫폼
                        → 균 자체 + 균이 만드는 물질 + 균 간 네트워크 + 숙주-미생물 신호축 모두 포함

1.2 패러다임 전환: 4단계

단계시기핵심 개념대표 소재
Phase 0~2015항균 (microbiome을 제거 대상으로 인식)항생제, 방부제, 살균 계면활성제
Phase 12015–2020마이크로바이옴 보호 (손상 최소화)마이크로바이옴-프렌들리 제형
Phase 22020–2024균형 회복 (Correction)프로바이오틱 라이세이트, Galactomyces 여과물

3. 마이크로바이옴 원료 세대별 분류 체계

1세대

발효 여과물·사균체 시대 (2015–2022)

특정 균주를 배양·발효 후 여과하거나 열처리·분쇄한 성분. '마이크로바이옴'이라는 소구 없이 '발효 성분'으로 마케팅된 경우도 다수.

원료명균주주요 효능
Galactomyces Ferment FiltrateGalactomyces candidum미백, 모공 축소, 피지 균형 — SK-II 핵심
Bifida Ferment LysateBifidobacterium bifidum장벽 강화, 항염, pH 조절
Lactobacillus FermentLactobacillus spp.항염, 진정, 피부 방어
Saccharomyces Ferment FiltrateSaccharomyces cerevisiae항산화, 세포 재생, 미백
Yeast Beta-Glucan효모 세포벽 유래TLR 활성화, 면역 조절, 장벽 강화
⚠️ 한계: 성분 조성 불명확, 배치 간 재현성 문제, '마이크로바이옴'이라는 기전적 주장 근거 부족.
2세대

포스트바이오틱 복합계 시대 (2022–2025)

비가열·특정 대사물 농축·이중 발효 등 고도화된 공정으로 특정 활성 성분(SCFA, 펩타이드, EPS 등)을 정량화하고, 단일 원료보다 복합 포뮬러를 구성.

포스트바이오틱의 구성 성분
─────────────────────────────────────────────────────
① 단쇄지방산 (SCFA): Propionic acid, Butyric acid, Acetic acid
   → 모낭 pH 조절, 항병원균, 장벽 세라마이드 합성 촉진
② 박테리오신 (Bacteriocin): Nisin, Lugdunin, Mersacidin
   → 선택적 항균 (병원성균만 억제), S. aureus 쿼럼센싱 억제
③ 세포외다당류 (EPS): 피부 수분막 형성, TLR4 조절, HA 유사 보습
④ 미생물 유래 펩타이드: 콜라겐 합성 자극, 항산화, 항균
⑤ 효소 (Enzyme): 각질 용해, 피지 분해, SOD·Catalase 유사체
⑥ 비타민·코팩터: B군 비타민(나이아신아마이드 전구체), CoQ10 유사체

대표 신규 사례 (2025):

  • Akkermansia muciniphila YGMCC2602 유래 포스트바이오틱 → 섬유아세포 증식, 콜라겐 합성, UV 손상 완화 (2025, ScienceDirect)
  • Staphylococcus hominis A9 발효 여과물 → S. aureus agr 시스템 억제 (AD 임상 Phase 2)
3세대

생태계 설계 플랫폼 시대 (현재 진입)

단일 균주나 단일 대사물이 아닌, 피부 마이크로바이옴 생태계 전체의 구조와 기능을 설계하는 플랫폼 기술.

3세대 기술의 3대 축
────────────────────────────────────────────────────
① 정밀 발효 (Precision Fermentation)
   → 합성생물학으로 균주를 최적화 설계
   → 원하는 활성 성분만 대량 고순도 생산
   → 지속가능성 + 동물 유래 성분 대체
② 세포외소포체 기반 (Bacterial/Microbial EVs)
   → 세균이 자연 방출하는 나노소포체(OMV, MV)
   → 피부-마이크로바이옴 간 신호전달 매개체
   → 면역 훈련, 항염, 재생 효능
③ 마이크로바이옴 생태계 설계 (Ecosystem Engineering)
   → AI + 메타게노믹스 기반 개인 피부 생태계 분석
   → 'soil'이 아닌 'ecosystem'을 디자인하는 개념

4.3 포스트바이오틱 (Postbiotic)

화장품 마이크로바이옴 원료의 현재 중심축

포스트바이오틱 3대 형태
────────────────────────────────────────────────────────────
① 발효 여과물 (Ferment Filtrate)
   → 균 배양 후 균체 제거, 상등액 농축
   → 대표: Galactomyces Ferment Filtrate, Saccharomyces Ferment
② 라이세이트 (Lysate / Tyndallizate)
   → 균체 파쇄 또는 열처리 후 세포 내용물 방출
   → 대표: Bifida Ferment Lysate, Lactobacillus Lysate
③ 특정 대사물 순수 정제
   → EPS, 박테리오신, SCFA, 미생물 유래 펩타이드
   → 대표: Nisin, Butyric acid, 균 유래 HA

2025년 핵심 신규 포스트바이오틱 성분:

  • Akkermansia muciniphila 유래 포스트바이오틱: 섬유아세포 증식↑, 콜라겐 합성↑, UV 방어, 피부 수분 증가
  • Engineered Bacteriocin + Endolysin 복합: 선택적 항균 + 마이크로바이옴 보존 시너지 (USPTO 특허)
  • Parabacteroides goldsteinii OMV: 건선 피부 염증 억제 (동물 모델 검증)

5. 주목받는 기전 및 효능 컨셉

5.1 TLR (Toll-Like Receptor) 신호 조절

마이크로바이옴 성분이 피부 면역의 '선천성 훈련(innate training)'에 관여하는 핵심 기전

TLR2 (그람양성균 세포벽 인식): LTA, PGN 자극
  → S. epidermidis LTA → TLR2 → NF-κB 적정 활성화 → AMP 분비 + Tight Junction 강화

TLR4 (LPS 인식, 그람음성균):
  → 일부 EPS는 TLR4 길항 → 항염 효과
  → 저용량 자극 → 면역 관용 형성

Dectin-1 (β-glucan 인식):
  → 효모·진균 β-glucan → Dectin-1 → 항진균 면역 + 장벽 강화

5.2 쿼럼센싱 (Quorum Sensing) 조절

  • Autoinducer 억제: S. hominis 유래 lugdunin → S. aureus agr 시스템 억제 → 독소 분비 차단
  • Anti-QS 식물 유래 소재: 커큐민, 갈산, 캡사이신 → QS 신호 분자 억제
  • C. acnes QS 조절: 건강 phylotype(I-1, III)이 병원성 phylotype(RT4/RT5)의 QS 억제

5.3 AhR (Aryl Hydrocarbon Receptor) 축

  • 인돌 (tryptophan 대사물): C. acnes, Lactobacillus 유래 → AhR 활성화 → Filaggrin 발현↑, 장벽 수복

  • 2024년 Experimental & Molecular Medicine: AhR 경로가 피부 장벽과 마이크로바이옴의 크로스토크 핵심 경로 확인

5.4 Microbiota-Skin Axis 확장

  • 장-피부 축 (Gut-Skin Axis): 구강 프로바이오틱·식이 → 장 마이크로바이옴 변화 → 피부 염증·피지·노화 간접 조절
  • 두피-피부 축: 두피 마이크로바이옴(Malassezia, S. capitis)이 얼굴 피부에 미치는 연쇄 영향
  • 피부-뇌 축 (Skin-Brain Axis): 2025년 신규 연구 — 안면 마이크로바이옴이 심리적 웰빙과 연관 → '스트레스 완화 마이크로바이옴 코스메틱' 소구 가능성

6. Next-Microbiome 원료 개발 Ideation

Ideation 방법론: 기존 1·2세대 마이크로바이옴 원료의 한계(안정성·기전 불명확·개인차·규제 불확실성)를 출발점으로, 3세대 기술 트렌드와 교차해 실현 가능성 높은 6개 방향을 도출
💡 Next-01  —  세균 유래 세포외소포체 (Bacterial OMV) — 마이크로바이옴 신호 나르는 나노캐리어 원료
배경

세균은 자연적으로 외막소포체(Outer Membrane Vesicle, OMV) 또는 막소포체(Membrane Vesicle, MV)를 방출한다. 이는 균-균, 균-숙주 간 신호를 전달하는 '마이크로바이옴의 언어'다. 2025년 Experimental Dermatology 리뷰에 따르면 피부 마이크로바이옴 유래 OMV는 면역 조절·장벽 강화·항염 소구에서 엑소좀과 경쟁할 최유력 차세대 바이오소재다.

핵심 메커니즘
균주 배양 → OMV 자연 방출 (직경 20–300 nm)
OMV 내용물: LPS 단편, 균주 특이적 단백질, RNA, 효소, 독소 모사체
  → TLR 리간드 포함 → 피부 선천 면역 훈련 (Trained Immunity)
  → 균주 특이적 항균 단백질 탑재 → 선택적 병원균 억제
  → EPS·대사물 탑재 → 유효성분 피부 세포 내 전달
원료 개발 방향
  • 피부 상재균 (S. epidermidis, C. acnes 건강 phylotype) 배양 → 규격화된 OMV 정제 (초고속 원심분리, 크기배제 크로마토그래피)
  • 염증 유발 성분(LPS 전장) 제거한 LPS-free OMV 또는 de-LPS 처리
  • Parabacteroides goldsteinii OMV: 건선 피부 염증 억제 동물 모델 근거 → 피부 장벽 염증 화장품 확장 검토
  • OMV에 원하는 활성 성분을 담는 엔지니어드 OMV — 합성생물학으로 OMV 조성 조절
✅ 차별점: 기존 식물 엑소좀·줄기세포 엑소좀 대비 대량 생산 가능 (세균 배양 → 원가 경쟁력) / '피부 마이크로바이옴의 자연 신호 물질'이라는 혁신적 소구 / 나노캐리어 딜리버리 플랫폼
⚠️ 리스크: 안전성 평가 (OMV 내 독성 성분 제거 공정 필수) / 규격화·배치 재현성 확보 / 소비자 소구 전략 필요
연계 참고: [[skin_microbiome_report §2.3]] [[파지 치료 (Phage Therapy)]] [[피부 오가노이드]]
💡 Next-02  —  정밀 발효 (Precision Fermentation) × 합성생물학 — 피부 마이크로바이옴 대사물의 무한 확장
배경

전통 발효는 균주가 자연적으로 만드는 성분에 의존한다. 정밀 발효는 CRISPR·대사공학으로 균주를 최적화해 원하는 성분을 원하는 양으로 생산한다. 동물 유래 성분 대체·지속가능성·고순도의 3가지 트렌드를 동시에 충족한다.

원료 개발 방향
① 피부 마이크로바이옴 특이적 대사물 대량 생산
   S. epidermidis가 소량 생산하는 항-UV SCFAs
   → 정밀 발효 균주로 산업적 규모 생산 가능화

② 피부 상재균 유래 펩타이드 생산
   C. acnes 유래 항균 펩타이드 → 유전자 클로닝 → 재조합 생산

③ 마이크로바이옴 관련 신규 보습제 생산
   균 유래 EPS → HA 유사 보습력 → 균주 배양 최적화로 고수율 EPS 생산

④ 동물 유래 성분 대체
   미생물 유래 콜라겐 (Geltor), 효모 발효 스쿠알렌

⑤ 희소 마이크로바이옴 대사물 공급
   Akkermansia 유래 Amuc_1100 단백질 → 정밀 발효로 공급 가능화
✅ 차별점: '마이크로바이옴 대사물'의 구조적 동일성 유지 + 무한 확장성 / 지속가능성·동물 프리 → ESG 소구 / 기존 화장품 규제 틀의 '발효 유래 성분'으로 분류 가능 → 규제 리스크 최소

기술 파트너 생태계: Geltor (콜라겐), StrainX Bioworks (정밀 발효 플랫폼), Ginkgo Bioworks (합성생물학 플랫폼)

💡 Next-04  —  엔지니어드 포스트바이오틱 — 박테리오신 + 엔도라이신 복합 정밀 원료
배경

2023년 USPTO 특허(US11633348, US12214070)에서 박테리오신 + 엔도라이신의 복합 포스트바이오틱이 단독 사용 대비 시너지 효과를 보임이 확인되었다. 마이크로바이옴 성분을 '자연에서 추출'하는 수준을 넘어, 원하는 기능을 가진 성분을 설계(engineering)하는 새로운 원료 패러다임이다.

핵심 성분
박테리오신 (Bacteriocin):
  → 미생물이 생산하는 항균 단백질/펩타이드
  → Nisin (Lactococcus lactis 유래): 그람양성균 광범위 억제
  → Lugdunin (S. hominis 유래): S. aureus 특이적 억제
  → Mutacin (S. mutans 유래): 구강·피부 적용 가능성

엔도라이신 (Endolysin):
  → 파지가 세균을 용균시키는 효소 → 세균 세포벽 분해
  → C. acnes, S. aureus 특이적 엔도라이신 식별 가능

복합 시너지:
  박테리오신 (막 퍼미아빌리티 증가) + 엔도라이신 (세포벽 분해)
  → 병원성균 이중 표적 → 내성 발생 가능성 극소화
제형 아이디어

앰플, 스팟 트리트먼트, 두피 에센스, 마이크로바이옴 밸런싱 세럼

✅ 차별점: 항생제 대체 + 마이크로바이옴 보존이라는 윤리적 소구 / 특허 가능성 높음 (IP화) / Phage therapy의 화장품 버전 포지셔닝

💡 Next-05  —  마이크로바이옴 친화 제형 설계 2.0 — 기제 성분까지 재설계
배경

BASF IFSCC 2025 충격적 결과 발표: 화장품의 비활성 기제 성분(방부제, 계면활성제, pH 조절제, 용매)도 피부 마이크로바이옴을 유의미하게 변화시킨다. '마이크로바이옴 성분'을 추가하는 것에서 벗어나, 제형 전체를 마이크로바이옴 생태계 관점에서 재설계해야 한다는 패러다임.

재설계 대상 성분 카테고리
① 방부 시스템 (Preservation)
   기존: Phenoxyethanol, Methylparaben, MIT → 마이크로바이옴 교란 가능
   대안: LACTOBACILLUS FERMENT (발효 방부제)
        → Glycolipid (미생물 유래 바이오서팩탄트)
        → 저pH 완충 시스템 (산 기반 방부)

② 계면활성제 (Surfactant)
   기존: SLS, SLES → 마이크로바이옴 다양성 파괴
   대안: 바이오서팩탄트 (Sophorolipid, Rhamnolipid)
        → 피부 친화적 + 마이크로바이옴 보존 + 생분해성

③ 용매·보습제
   기존: 부틸렌글리콜, 프로필렌글리콜
   대안: 균 유래 EPS, 발효 글리세롤, 프리바이오틱 보습제

④ pH 조절
   기존: 인공 pH 조절제
   대안: 유기산 발효 여과물 기반 완충 시스템
✅ 차별점: 기존 마이크로바이옴 화장품과의 근본적 차별화 — '성분을 추가'하는 것이 아닌 '제형 자체를 마이크로바이옴-설계' / 마이크로바이옴 친화 지수(MFI) 지표화 → 브랜드 인증·소구 수단

💡 Next-06  —  멀티-축 마이크로바이옴 전략 — 장-피부-뇌 축 크로스오버 원료
배경

피부 마이크로바이옴은 독립적으로 존재하지 않는다. 장-피부 축(Gut-Skin Axis), 피부-뇌 축(Skin-Brain Axis)이 체계적으로 연결되어 있으며, 통합 전략이 차세대 마이크로바이옴 원료의 방향이다.

3대 축 전략
[Inside-Out: 경구 → 장 → 피부]
구강 섭취 프로바이오틱/프리바이오틱
  → 장 마이크로바이옴 조성 → 장 투과성 정상화
  → 전신 염증 감소 → 피부 염증·피지·노화 개선
  예시: Lactobacillus reuteri 경구 투여 → 피부 세라마이드↑ (2024)

[Outside-In: 피부 도포 → 국소 생태계 → 전신 신호]
특정 상재균 지원 포스트바이오틱 도포 → 모낭 마이크로바이옴 정상화
  → 국소 면역 훈련 → 전신 알레르기 반응↓
  예시: Roseomonas mucosa LBP → AD 피부 도포 → TH2 억제 임상 근거

[Skin-Brain Axis: 피부 마이크로바이옴 → 신경계 → 웰빙]
2025 신규 연구: 안면·두피 마이크로바이옴 조성 → 심리적 웰빙과 상관
  → '스트레스 피부' 케어의 새로운 과학적 근거
  예시: 마이크로바이옴 균형 → Substance P 조절 → 피부 스트레스 반응↓
원료 개발 방향
  • Oral-Topical 시너지 패키지: 경구 프로바이오틱 + 토포 포스트바이오틱 세트 개발 → 인사이드-아웃 효과 시너지 임상 설계
